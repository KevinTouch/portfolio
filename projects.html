YelpCamp
About: YelpCamp is a website which allows users to sign up and be able to post pictures of campsites, that they have been before. Futhermore, users will be able to post, edit, and delete comments on these campsites forums. What I learned: From my experience, this project taught me RESTFUL api behavior (e.g. request/response) and the different HTTP request methods (POST, GET, UPDATE, DELETE, etc) when it came to creating api endpoints. In terms of skillset, I was able to familiarize myself with using node js, express, passport and other npm libaries that I have not used in the past. Lastly, I learned about using mongodb (which was my first non-relational db) in conjunction with express to implement data persistence in web.
